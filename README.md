# Locker Contract
One night Mob programming project using truffle pet-shop box as started point.


## Requirements
- Ganache
- Truffle


## Dev
```bash
truffle compile
truffle migrate
truffle test
truffle dev
```

## DApp
start by type `npm run dev`.

- index.html - main page
- admin.html - contract setting
- locker.html - mimic Physical Locker
